# MovieTalk

TMDB OpenAPI를 활용한 영화 정보, 리뷰, 추천 플랫폼
실무형 CRUD, 인증, REST API, DB설계, 배포, 관리자 기능까지 구현


사이트 주소: http://movitalk.cloud

## 📝 주요 기능
- 회원가입/로그인
- 영화 리뷰/댓글/별점/좋아요/신고
- 인기영화 집계(리뷰수/조회수 기준 TOP-N)
- 개인화 추천/온보딩
- 배우 검색/필모그래피/댓글
- 마이페이지(내역, 정보관리)
- 관리자(회원/리뷰/신고/통계 관리)
